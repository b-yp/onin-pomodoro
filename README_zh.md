# Onin Pomodoro 🍅

[English](README.md) | 简体中文

为 Onin 设计的一款简洁、高效的番茄钟插件，旨在帮助你轻松保持专注并追踪工作时间。

![App Screenshot](app.png)

## 功能特点

- **自定义时长**：灵活设置专注、短休息和长休息的时间间隔。
- **阶段管理**：在专注、短休息和长休息阶段之间无缝切换。
- **持久化运行**：即使关闭插件窗口，计时器仍会在后台继续运行。
- **智能休息提醒**：在完成一定次数（默认为 4 次）的专注后，自动建议长休息。
- **系统通知**：当专注或休息结束时，立即收到系统提醒。
- **专注历史**：通过交互式柱状图可视化展示过去 14 天的生产力数据。
- **每日统计**：追踪今天收集到的“番茄”数量。
- **高级设计**：采用现代毛玻璃 UI，拥有流畅的动画和径向渐变背景。

## 安装

这是 [Onin](https://onin-app.github.io/Onin/) 的一个插件。安装步骤：

1. 打开 Onin。
2. 进入“插件”部分。
3. 导入或安装 "Onin Pomodoro"。

## 开发

如果你想参与贡献或从源码构建插件：

### 环境准备

- [Node.js](https://nodejs.org/)
- [pnpm](https://pnpm.io/)

### 常用命令

```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm run dev

# 构建插件
pnpm run build
```

## 权限说明

此插件需要以下权限才能正常运行：
- `scheduler`：用于后台计时器执行。
- `storage`：用于保存设置和专注历史。
- `notification`：用于发送阶段完成提醒。
